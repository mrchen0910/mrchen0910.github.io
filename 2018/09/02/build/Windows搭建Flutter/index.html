Windows平台下搭建Flutter

1、下载安装Fluttter SDK

在flutter的官网上下载最新的flutterSDK压缩包，解压在你的本地磁盘下；解压后会在flutter的文件夹下看到bin 目录，将该路径配置到系统的环境变量当中。

2、安装Dart和Flutter插件

1）启动Android Studio
2） 打开插件面板如下图所示
3）选择【Browse repositories…】，勾选Flutter plug-in 然后点击 install
4）当提示是否安装Dart插件时点击Yes
5）安装完成后重启AS
